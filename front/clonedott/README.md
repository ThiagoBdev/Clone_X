# Clone X

Projeto feito com React e TypeScript. Futuramente terá integração com Python.

## Tecnologias

- React / Next.js
- TypeScript
- Node.js
- Python (futuro)

## Instalação

1. Clone o repositório:

   git clone https://github.com/ThiagoBdev/Clone_X.git

2. Instale as dependências:

npm install

3. Rode o Projeto:

npm run dev
